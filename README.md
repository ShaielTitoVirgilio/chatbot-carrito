# 🍔 Chatbot para Local de Comidas Rápidas – Paysandú

Este repositorio contiene un chatbot diseñado para automatizar la atención al cliente en un local de comidas rápidas ubicado en Paysandú, Uruguay.

## 📌 ¿Qué hace este chatbot?

El sistema permite interactuar con los clientes de forma automática, brindando una experiencia ágil y eficiente. Sus principales funciones son:

* 🧾 Tomar pedidos completos de los clientes
* 💬 Mantener una conversación natural (consultas, menú, precios, etc.)
* 📦 Procesar y organizar los pedidos realizados
* 🔁 Derivar consultas específicas o situaciones fuera de su alcance al personal humano

## ⚙️ ¿Cómo funciona?

El chatbot utiliza la información disponible (menú, precios, horarios, etc.) para responder a los usuarios y guiarlos durante el proceso de compra. Una vez que el pedido está completo o si surge una consulta más compleja, el sistema envía la información al personal del local para su seguimiento.

## 👨‍🍳 Objetivo

Optimizar la atención al cliente, reducir tiempos de espera y facilitar la gestión de pedidos en el día a día del local.
